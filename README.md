# Images Converter
**Professional Browser-Based Image Processing Engine**

Images Converter adalah alat berbasis web yang memungkinkan pengguna untuk melakukan konversi format gambar secara instan tanpa perlu mengunggah file ke server. Seluruh proses konversi dilakukan langsung di sisi klien (Client-side), menjamin privasi dan kecepatan akses yang maksimal.

<img src="https://raw.githubusercontent.com/Mister-RdanM19/Image.Converter/refs/heads/main/image.png">


## Fitur Utama
- **Privasi Terjamin:** Gambar tidak pernah dikirim ke server; semua pemrosesan terjadi di dalam browser pengguna.
- **Dukungan 20+ Format:** Mengonversi antar format populer termasuk PNG, JPG, WebP, ICO, BMP, TIFF, hingga ekspor ke PDF.
- **Antarmuka Profesional:** Desain responsif dengan mode gelap (Dark Mode) yang bersih dan fitur Drag & Drop.
- **Tanpa Instalasi:** Berjalan langsung di browser melalui GitHub Pages.

## Teknologi yang Digunakan
- **HTML5 & CSS3:** Struktur dan styling menggunakan standar modern.
- **JavaScript (Vanilla):** Logika pemrosesan gambar menggunakan Canvas API.
- **UTIF.js:** Library tambahan untuk menangani format gambar tingkat lanjut seperti TIFF.
- **GitHub Pages:** Infrastruktur hosting statis yang cepat.

## Cara Penggunaan
1. Buka aplikasi melalui link GitHub Pages.
2. Tarik file gambar yang ingin dikonversi ke area drop-zone atau klik untuk memilih file dari komputer.
3. Pilih format tujuan pada menu dropdown yang tersedia.
4. Klik tombol "Konversi Sekarang".
5. File hasil konversi akan terunduh secara otomatis ke perangkat Anda.

## Daftar Format yang Didukung
- **Web:** PNG, JPG, JPEG, WebP, GIF, APNG.
- **System & Icons:** ICO, CUR, BMP.
- **Professional & Docs:** TIFF, TIF, PDF (via print stream), SVG.
- **Lainnya:** AVIF, JFIF, PJP, TGA, dan berbagai variasi bitmap lainnya.
